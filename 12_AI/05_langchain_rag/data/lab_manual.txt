[실습실 장비 및 장애 대응]
교육용 노트북의 공용 계정은 student이며 초기 비밀번호는 class2026이다.
수업 시작 후에는 개인 비밀번호로 반드시 변경해야 한다.

와이파이 이름은 SONGPA-AI이고 비밀번호는 ai-study-2026이다.
와이파이에 연결되지 않으면 비행기 모드를 10초 동안 켰다가 끈다.
문제가 계속되면 강사 PC 옆의 유선 어댑터를 사용한다.

OpenAI API 키는 코드에 직접 작성하거나 Git 저장소에 올리면 안 된다.
API 키는 프로젝트의 .env 파일에 OPENAI_API_KEY 이름으로 저장한다.
.env 파일은 .gitignore에 추가해 외부에 공개되지 않도록 한다.
