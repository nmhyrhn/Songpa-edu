3. 결론: 가장 안전한 방법은 '배열'로 바꾸는 것!

        `querySelectorAll`은 이런 문제가 없는 '고정된' `NodeList`를 반환하기 때문에 훨씬 안전합니다.
        하지만 만약 `getElementsBy...` 같은 메서드를 꼭 써야 한다면,
        가져온 목록을 **먼저 진짜 배열로 변환한 후 사용하는 것이 가장 안전합니다.**
