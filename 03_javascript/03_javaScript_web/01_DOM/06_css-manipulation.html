CSS 조작: 인라인 스타일 vs 클래스 제어

1. 임시 명찰 붙이기 (인라인 스타일: element.style)

    `element.style` 프로퍼티를 사용하면, 해당 요소에 '인라인 스타일'을 직접 추가하거나 변경할 수 있습니다.
    간단한 동적 효과를 줄 때 유용하지만, 스타일 관리가 복잡해진다는 단점이 있습니다.
첫 번째 박스

2. '직업' 부여하기 (클래스 제어: element.classList)

    스타일은 CSS 파일에 미리 정의해두고, 자바스크립트에서는 클래스를 추가/제거/교체하는 것이
    가장 현대적이고 효율적인 방법입니다. `element.classList`를 사용합니다.
